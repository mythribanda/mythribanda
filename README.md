**Technologies:** React · TypeScript · JavaScript · CSS Grid · Node.js · AI/LLM Integration · Sandboxed Execution

## 🧠 What I've Learned Building Products

<details>
<summary><b>Key Lessons from Shipped Projects</b></summary>

### **From PillTalk (Healthcare AI)**
- Precision matters in healthcare—edge cases can affect real users
- Combining multiple AI modalities (vision + NLP + LLM) requires careful orchestration
- Accessibility and user education are as important as core functionality

### **From EduSim (EdTech Platform)**
- React's complexity scales—proper state management and architecture choices made the difference
- RAG systems need fallback mechanisms for robustness
- Educational content requires iterative refinement based on user feedback

### **From Data Whisperer (Analytics)**
- Automating complex workflows requires understanding domain edge cases deeply
- Data quality and cleaning are 80% of the work in analytics
- Visualization and story-telling matter more than raw accuracy

### **From Competitive Programming Projects**
- Algorithmic efficiency matters—especially for large datasets
- Understanding time/space tradeoffs is crucial
- Problem decomposition is the most valuable skill

### **Cross-Project Lessons**
- ✅ **Ship early** – Get feedback from real users as soon as possible
- ✅ **Measure impact** – Define success metrics before building
- ✅ **Iterate relentlessly** – First version is never the final version
- ✅ **Document thoroughly** – Future you (and your team) will thank you
- ✅ **Automate repetition** – Tools and scripts compound into massive time savings
